ATAQUE DE FUERZA BRUTA
📌 DESCRIPCION 
Este programa implementa un ataque de fuerza bruta que intenta encontrar una contraseña probando todas las combinaciones posibles.
COMO EJECUTAR
python bruteforce.py
Luego ingresar una contraseña (máximo 4 letras).
💻 EJEMPLO DE SALIDA
Ingresa la contraseña máximo 4 letras: ab
Contraseña encontrada felicidades : ab
Intentos realizados: 28
Tiempo: 0.0012 segundos
📊 GRAFICO
El programa genera un gráfico donde:
Eje X: longitud de la contraseña
Eje Y: número de intentos
🧠 REFLEXION
Si la contraseña tiene más de 8 caracteres y usa mayúsculas, números y símbolos:
El número de combinaciones crece exponencialmente
El tiempo requerido aumenta demasiado
El ataque de fuerza bruta se vuelve poco práctico
Esto demuestra la importancia de usar contraseñas seguras.
